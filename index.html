Características y Ventajas de Apache

  • Servidor web HTTP de código abierto.
  • Desarrollado dentro del proyecto HTTP Server (httpd).
  • Implementa el protocolo HTTP/1.1.
  • Modular y multi-plataforma (Windows, Linux, MacOS).
  • Muy estable y con soporte amplio.

Estructura del Servidor Apache

Apache utiliza una estructura bien definida para su funcionamiento:

  • Directorios de contenido: Contienen las aplicaciones web del servidor.
  • Directorios de registros: Archivos log como /var/log/apache2/access.log.
  • Directorios de configuración: Archivos como /etc/apache2/apache2.conf.
Directorios importantes
  • /var/www: Contenido web.
  • /var/log/apache2: Registros de actividad.
  • /etc/apache2: Configuración global.

Directivas Básicas

Configuraciones esenciales para Apache:

Directiva Descripción Ejemplo
ServerName Nombre del servidor. ServerName daw27.deap27.net
DocumentRoot Directorio principal de contenido. DocumentRoot /var/www/html
ErrorLog Archivo log de errores. ErrorLog /var/log/apache2/error.log

Directivas de Bloque

Las directivas se agrupan en bloques para un control granular:

Controla las opciones disponibles en un directorio. 
<Directory /ruta>
    Options All
    AllowOverride None
</Directory>

Controla las opciones para ficheros específicos. 
<Files "archivo.html">
    Require all granted
</Files>
